# 👨‍🏫 助教评审说明

## 🎯 项目概述

**AI旅行规划师** - 一个功能完善、界面现代的智能旅行规划Web应用

- 🤖 **AI驱动**：使用阿里云百炼大模型生成个性化行程
- 🎤 **语音识别**：支持自然语言语音输入
- 🗺️ **全球地点**：支持国内外旅行地点（四层智能地理编码）
- 💰 **预算管理**：实时追踪和AI分析建议
- 🎨 **现代UI**：深色模式、玻璃态设计、流畅动画

## ⚡ 快速部署（5分钟）

### 方式一：使用Docker（推荐）

```bash
# 1. 拉取镜像
docker pull crpi-6mppr7h5m3b03dk6.cn-hangzhou.personal.cr.aliyuncs.com/aitravel/ai-travel-planner:latest

# 2. 克隆项目（获取配置文件）
git clone https://github.com/zhulongqihan/ai-travel-planner.git
cd ai-travel-planner

# 3. 配置环境变量
cp env.docker.example .env.docker
# 编辑.env.docker，填入您的阿里云百炼API Key

# 4. 启动服务
docker run -d -p 8000:8000 --env-file .env.docker \
  --name ai-travel-planner \
  crpi-6mppr7h5m3b03dk6.cn-hangzhou.personal.cr.aliyuncs.com/aitravel/ai-travel-planner:latest

# 5. 访问应用
# 浏览器打开 http://localhost:8000
```

### 方式二：本地运行

```bash
# 1. 克隆项目
git clone https://github.com/zhulongqihan/ai-travel-planner.git
cd ai-travel-planner

# 2. 创建虚拟环境
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/Mac

# 3. 安装依赖
pip install -r requirements.txt

# 4. 配置环境变量
cp .env.template .env
# 编辑.env文件

# 5. 运行服务
python run.py

# 6. 访问应用
# 浏览器打开 http://localhost:8000
```

1. **国际路线规划**：国际地点仅支持标记，不支持路线规划（高德地图限制）
2. **语音识别**：需要HTTPS或localhost环境
3. **浏览器兼容性**：推荐Chrome 90+、Edge 90+
