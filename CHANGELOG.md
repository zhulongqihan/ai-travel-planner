# 更新日志

本项目的所有重要更改都将记录在此文件中。

## [1.0.0] - 2024-01-15

### 新增

#### 核心功能
- ✨ 智能行程规划功能
  - 支持语音输入旅行需求
  - 支持文字输入旅行参数
  - 使用阿里云百炼 AI 生成个性化行程
  - 详细的每日行程安排（景点、餐饮、住宿）
  - 交通方式建议
  - 费用预估

- 💰 费用预算与管理
  - 预算分析和估算
  - 实际费用记录（支持语音输入）
  - 预算对比和跟踪
  - AI 智能预算建议

- 👤 用户管理与数据存储
  - 用户注册登录系统
  - 基于 Supabase 的云端存储
  - 多设备数据同步
  - 行级数据安全

- 🗺️ 地图导航
  - 高德地图集成
  - 景点位置标记
  - 路线规划功能

#### 技术实现
- FastAPI 后端框架
- Supabase 数据库和认证
- 阿里云百炼大语言模型
- 阿里云语音识别 API
- 高德地图 API
- 响应式前端设计

### 文档
- 📖 完整的 README 文档
- 📋 详细的 API 文档
- ⚙️ 配置指南
- 🚀 部署指南
- 🤝 贡献指南

### 安全性
- 🔒 环境变量管理
- 🔐 API 密钥安全存储
- 🛡️ 行级安全策略
- ✅ CORS 配置
